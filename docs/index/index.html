Module index

This module provides functionality for managing and searching a Lucene search index

Example 

var {Index} = require("lucindex");
var {Document, Field} = org.apache.lucene.document;
var index = Index.createRamIndex();
var doc = new Document();
doc.add(new Field("id", 1,
            Field.Store.YES, Field.Index.ANALYZED, Field.TermVector.NO))
// add the document to the index - this is done asynchronously!
index.add(doc);

Index.createIndex (dir, name, analyzer, forceCreate)

Creates an index stored on disk

Parameters

String dir The directory to store the index in
String name The name of the index. The index files will be stored in a subdirectory with this name with the base directory passed as first argument.
org.apache.lucene.analysis.Analyzer analyzer The analyzer to use for indexing
Boolean forceCreate If true the index will be emptied if already existing

Index.prototype.close ()

Closes the index writer. Normally it should never be necessary to call this method manually, as the index writer will be closed when all index manipulation jobs have been finished.

Index.prototype.remove (name, value)

Removes all documents that match the given name/value pair from the index. Removal is done asynchronously.

Parameters

String value The value of the document field

Index.prototype.update (name, value, doc)
